# Markdown Wiki

## Install

    npm init
    npm install --save @schornio/md-wiki

Add `"start": "md-wiki"` to `package.json`

    {
      ...
      "scripts": {
       "start": "md-wiki",
       ...
      },
      ...
    }


## Start

    npm start

## Error handling

For custom 404 error handling add `error/404.md`
